# Constant

---

A class holding constant values used by the library.

| Name | Description |
|:-----|:------------|
| DefaultBatchOperationSize | The default batch size for batch operations ([InsertAll](/operation/insertall), [MergeAll](/operation/mergeall), [UpdateAll](/operation/updateall)). |
| DefaultCacheItemExpirationInMinutes | The default cache expiration in minutes, used by the [CacheItem.ExpirationInMinutes](/class/cacheitem) property. |
